边框样式应用

  1. none：默认值，无边框，不受任何指定的border-width值影响。
  2. hidden：隐藏边框，IE不支持。
  3. dotted：定义点线。
  4. dashed：定义虚线。
  5. solid：定义实线。
  6. double：定义双线边框，两条线及其间隔宽等于指定的border-width值。
  7. groove：根据border-color值定义D凹槽。
  8. ridge：根据border-color值定义D凸槽。
  9. inset：根据border-color值定义D凹边。
  10. outset：根据border-color值定义D凸边。
